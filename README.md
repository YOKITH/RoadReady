# 🚗 RoadReady

Car Rental Management System built using:

- ASP.NET Core Web API
- SQL Server
- Entity Framework Core
- JWT Authentication
- AutoMapper
- FluentValidation
- NUnit & Moq
- React (Frontend - In Progress)

## Project Structure

RoadReady.API      -> Backend API

RoadReady.Tests    -> Unit Tests

RoadReady.UI       -> React Frontend (Coming Soon)
